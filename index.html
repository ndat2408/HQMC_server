Hướng dẫn tham gia vào game

I. Tải TLauncher, Java

  • 1. Truy cập vào trang chủ của TLauncher để tải Launcher hoặc bấm vào đây để tải ngay bản cho windows
  • 2. Sau khi tải xong thì cài đặt như bình thường rồi mở launcher và làm tiếp theo hướng dẫn bên dưới
  • 3. Nếu gặp lỗi như hình thì bạn tiến hành tải Java ở trang chủ hoặc bấm vào đây để tải bản 64 bit hoặc 32 bit

II. Tham gia server

  • 1. Mở TLauncher, nhập tên ingame của bạn, chọn 1 trong 5 bản 1.15.2 (nên chọn bản OptiFine) sau đó chọn install, đợi game tải xong thì chọn "Enter the game"
  • 2. Sau khi vào game, chọn Multiplayer (Chơi Mạng)
  • 3. Nếu nó hiện như thế này thì bạn click vào ô "Do not show" sau đó bấm Proceed
  • 4. Sau đó chọn Add server (Thêm máy chủ)
  • 5. Tại dòng 2, điền THPTHQ.aternos.me sau đó bấm done
  • 6. Sau khi xong, server sẽ hiện và bạn chỉ cần bấm vào là có thể vào được server và khám phá rồi. Chúc bạn chơi game vui vẻ

Hướng dẫn tương tác trong game

  • - Dùng W, A, S, D để di chuyển
  • - Bấm Ctrl trong lúc di chuyển để chạy nhanh
  • - Xoay chuột để xoay tầm nhìn
  • - Dùng chuột phải để thao tác với game (Đóng/Mở cửa)
  • - Dùng chuột trái để đánh người khác
  • - Bấm T để chat all
  • - gõ /tell username để chat riêng với bạn của bạn (thay username bằng tên bạn của bạn)
  • - Giữ C để zoom
  • - Bấm E để mở túi đồ
  • - Bấm F1 để ẩn HUD (tay, khay đồ)
  • - Bấm F5 để đổi góc nhìn (Nên dùng góc nhìn thứ 3)
