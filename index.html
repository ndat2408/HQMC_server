II. Tham gia server

  1. Mở TLauncher chọn "Enter the game"
  2. Chọn Chơi Mạng (Multiplayer)
  3. Chọn Thêm server (Add server)
  4. Tại dòng 2, điền ........ sau đó bấm done
  5. Sau khi xong thì vào phá thôi

Hướng dẫn trong game

  • Dùng W, A, S, D để di chuyển
  • Bấm Ctrl trong lúc di chuyển để chạy nhanh
  • Xoay chuột để xoay tầm nhìn
  • Dùng chuột phải để thao tác với game (Đóng/Mở cửa)
  • Dùng chuột trái để đánh người khác
  • Giữ C để zoom
  • Bấm E để mở túi đồ
  • Bấm F1 để ẩn HUD (tay, khay đồ)
  • Bấm F5 để đổi góc nhìn (Nên dùng góc nhìn thứ 3)
